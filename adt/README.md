# Aria Digital Twin Dataset Tools

This project provides tools for predicting the next object a user might interact with, based on spatial context and interaction history.

## Prerequisites

1. **Python**: Ensure you have Python 3.7, 3.9, 3.10 or later installed.
2. **Environment Variables**: Set the `OPENAI_API_KEY` environment variable with your OpenAI API key.

   ```bash
   export OPENAI_API_KEY="your-api-key-here"
